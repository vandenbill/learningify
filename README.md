## Description

Learningify App, Made to Participate the 2022 Telkom SECS Competition

## Installation

```bash
$ npm install
```

## API Documentation

```bash
# swagger api documentation
{baseUrl}/api
```

## Setup Database

```bash
# migrate database
npx prisma migrate

# launch prisma studio
npx prisma studio
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
